Ngee Ann Polytechnic

Ngee Ann City: A City-Building Strategy Game

Appendix A: IT Project

1. BACKGROUND

Develop a city-building strategy game called Ngee Ann City.

2. SCOPE

The player is the mayor of Ngee Ann City, and the goal of the game is to build the happiest and most prosperous city possible, i.e., score the most points.

There are 2 game modes, one Arcade mode with a limited number of coins and grid, while the other is Free Play mode with unlimited coins and grid.

In Arcade mode, this city-building game begins with 16 coins. In each turn, the player will construct one of two randomly selected buildings in the 20x20 city. Each construction costs 1 coin...

3. GAME REQUIREMENTS

The game should provide the following features:

  • Display main menu
  • Start New Arcade Game
  • Start New Free Play Game
  • Load Saved Game
  • Display High Scores
  • Exit Game
